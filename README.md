# Simon-Game
🎮 Simon Game A simple memory-based game built using HTML, CSS, JavaScript, and jQuery.  Watch → Remember → Repeat the color sequence Make one mistake... and it's Game Over! 😱

**Features**
Dynamic color pattern generation

User interaction through button clicks

Sound effects for each color

Button press animation

Game over feedback and restart on key press

Level-based difficulty increase

🛠️ **Tech Used**
HTML5

CSS3

JavaScript

jQuery

🚀 **How to Play**
Press any key to start the game.

A random color button will flash with sound.

Click the buttons in the same order.

Each new level adds one more color to the sequence.

Match the pattern correctly to advance.

One wrong click = Game Over. Start again!
